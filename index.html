Seja bem-vindo à nossa plataforma de ensino do Método Scrum!

A equipe CodeDontBlow veio proporcionar um site que ensinará passo a passo o que é o método Scrum, com uma trilha de aprendizagem dinâmica, exemplos práticos e testes para verificar a sua compreensão. Nosso site foca principalmente em entregar uma boa experiência e a melhor compreensão do aprendizado Scrum. A seguir, explicaremos como funciona cada seção do nosso site.

Trilhas de conhecimento:

As trilhas de conhecimento foram estudadas e desenvolvidas em uma ordem de conteúdo para que você estude e compreenda o Scrum de forma dinâmica e eficiente, sendo que cada componente é encaixado em um lugar específico para fazer sentido. Ela é dividida em seções do Método Scrum e cada seção possui páginas com os elementos correspondentes, no final de cada seção, você pode acessar um quiz para verificar o seu conhecimento, seu desemmpenho nesse quiz não determiná se vc foi aprovado ou não.

Seção 1: Definições e Origem

  • Fundamentos do Scrum;
  • Manifesto ágil;
  • Pilares do método;
  • Quiz de conhecimento - Definições e origem;

Games interativos:

Os games interativos são baseados em situações presentes no método Scrum. Foram desenvolvidos para proporcionar uma experiência prática de como é trabalhar com esse método, permitindo que você adquira uma noção básica do que fazer em cada situação. Assim, você não terá dificuldades ao aplicar o método na vida real. Os games estão disponíveis na página "Games Interativos", onde cada um possui um link junto com sua descrição. Basta clicar no game escolhido para iniciá-lo.

Game 1: Desenvolva seu Product Backlog

Um cliente fictício irá fornecer os requisitos e você irá montar o seu Sprint Backlog com base no que foi ensinado no curso, incluindo 3 User Stories por sprint, juntamento com seus respectivos dados
