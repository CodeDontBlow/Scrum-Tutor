Cerimônias Scrum

Sprint Review

A Sprint Review é uma importante etapa após o fim de uma Sprint no desenvolvimento ágil. Aqui, o Product Owner avalia o que foi entregue comparando com o planejado durante a Sprint Planning Meeting. Esta cerimônia é aberta tanto para o cliente quanto para outras partes interessadas.

Durante a Sprint Review, o Time Scrum apresenta claramente o que foi produzido e está disponível para responder a quaisquer perguntas dos presentes. Isso permite uma verificação conjunta do progresso e ajuda a garantir que as expectativas do cliente e do time estejam alinhadas.

Sprint Retrospective

A Sprint Retrospective é uma reunião crucial que marca o final de uma Sprint. Nela, todo o Time Scrum se reúne para analisar o que aconteceu durante a Sprint que acabou de terminar, discutidindo o que deu certo, identificando quais foram os desafios enfrentados pela equipe, sugerindo soluções para esses problemas e decidindo como podemos melhorar na próxima Sprint.

Release Planning Meeting

O Release Planning Meeting é uma reunião importante realizada pelo Time Scrum para planejar o lançamento de uma nova versão do produto, chamada de Release . Durante essa reunião, o time trabalha para definir a visão de longo prazo do projeto e estabelecer o caminho a ser seguido após uma Sprint. Isso inclui determinar um prazo estimado para a implementação das funcionalidades necessárias para o produto.
